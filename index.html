📋 FBM 配送費估算說明
國際快遞 DDP（含稅到門）：台灣直發到美國買家，由 DHL/FedEx/UPS 或第三方貨代處理清關和配送。費用含國際運費 + 燃油附加費 + 清關 + 關稅 + 最後一哩配送。
海外倉代發：先將商品批量海運到美國第三方倉庫（頭程 $1-2/kg），再由倉庫逐件出貨給買家（美國境內配送 $3-6/件）。總成本較低，但需預付頭程和倉租。

💡 FBM vs FBA
FBM 無需支付 FBA 配送費和倉儲費，但需自行處理物流、客服和退貨。商品不會有 Prime 標誌，可能影響曝光和轉換率。適合大型/重型商品、客製化商品或利潤較薄的品項。

📋 計算說明
此模式幫助台灣賣家評估：將現有台灣產品拿到 Amazon 美國站用 FBM 自配送銷售，會額外產生多少成本。

額外成本 = 銷售佣金 + 帳戶月費分攤 + FBM 國際配送 + 關稅 + 包裝客服

「建議美金售價」是根據你在台灣的利潤率，反推在 Amazon 上需要定多少價才能維持相同利潤率（已包含所有 Amazon 額外成本）。
